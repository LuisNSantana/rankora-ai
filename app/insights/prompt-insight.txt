# Base prompt for Business Insights generation

Analyze the following data in the {{type}} domain and generate an executive summary, key metrics, actionable recommendations, and suggested visualizations. The output must be structured and useful for business decision-making.

Data:
{{data}}

Analysis objective:
{{objective}}

Response format:
- summary: executive summary
- metrics: list of key metrics
- recommendations: list of actionable recommendations
- visualizations: suggested charts/tables
- sources: data sources
- meta: any other relevant information
